# Новые возможности аналитики - Геолокация и карта

## Что добавлено:

### 1. Расширенная геолокация
- ✅ Страна и код страны
- ✅ Регион/область
- ✅ Город
- ✅ Координаты (широта/долгота)
- ✅ Провайдер (ISP)
- ✅ Организация (Org)
- ✅ ASN номер

### 2. Интерактивная карта
- ✅ Карта с маркерами посещений
- ✅ Попапы с детальной информацией об IP
- ✅ Автоматическое масштабирование карты

### 3. Детальная информация в таблице
- ✅ Полная информация о местоположении
- ✅ Информация о провайдере
- ✅ ASN номер
- ✅ Координаты

## Источник данных

Используется бесплатный API **ip-api.com**:
- До 45 запросов/минуту без регистрации
- До 1000 запросов/день с регистрацией (опционально)
- Подробная информация об IP адресах

## Структура БД

Новые поля в таблице `visits`:
- `country_code` - код страны (RU, US, etc.)
- `region` - регион/область
- `latitude` - широта
- `longitude` - долгота
- `isp` - интернет-провайдер
- `org` - организация
- `asn` - ASN номер

## Миграция существующей БД

При запуске контейнера новые поля автоматически добавляются в существующую БД через `ALTER TABLE`.

## Пример данных

```
IP: 46.53.254.123
Страна: Belarus
Код: BY
Регион: Minsk Region
Город: Minsk
Координаты: 53.9045, 27.5615
Провайдер: Beltelecom
ASN: 6697
```

## Карта

Карта использует OpenStreetMap через библиотеку Leaflet.js:
- Бесплатно
- Не требует API ключей
- Работает офлайн (после загрузки)

## Ограничения

- Локальные IP адреса (127.0.0.1, 192.168.x.x) не определяются
- Лимит API: 45 запросов/минуту (можно увеличить с регистрацией)
- Точность зависит от провайдера (обычно до города)
